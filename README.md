# Flower Classification using Transfer Learning

## Overview
This project uses TensorFlow and EfficientNet Transfer Learning to classify flower images into five categories:

- Roses
- Daisy
- Dandelion
- Sunflowers
- Tulips

## Technologies Used
- Python
- TensorFlow
- TensorFlow Hub
- OpenCV
- NumPy
- Scikit-Learn

## Model
EfficientNet Lite0 Feature Extractor + Dense Classification Layer

## Workflow
1. Download Flower Dataset
2. Image Preprocessing
3. Train-Test Split
4. Transfer Learning
5. Model Training
6. Model Evaluation

## Dataset
TensorFlow Flower Photos Dataset
